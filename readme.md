# Gemini Cli 安裝
## 安裝 NodeJs
### Windows
1. 下載安裝檔並執行 `https://nodejs.org/en/download`
2. ``` shell
   node -v
   ```
### MacOs
1. `brew install node`
2. ``` shell
   node -v
   ```
### Linux
1. `sudo apt install nodejs`
2. ``` shell
   node -v
   ```
## 安裝 Gemini Cli
``` bash
npm install -g @google/gemini-cli
gemini
```
